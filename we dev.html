Introduction

Graphic design is a craft where professionals create visual content to communicate messages. By applying visual hierarchy and page layout techniques, designers use typography and pictures to meet users' specific needs and focus on the logic of displaying elements in interactive designs to optimize the user experience. Web development is a process of designing, building, testing, and maintaining a website which ranges from a simple single page static website to a complex full-stack applications. This field can be divided in several categories such as front end development, backend development,and fullstack development. In this course, we will cover all the important technologies and tools necessary for web development. DevOps is also another emerging field that came recently.
